Descrição:

    Fake JSON local server para execução de testes com aplicação de emprestimos a clientes.
    
    Informações dos clientes e tabela de taxas contidas no arquivo:
        localDB.json
    

Dependencias:

    JSON-Server (https://github.com/typicode/json-server)


Inicializar:

    Para inicializar o JSON Server, execute o comando:

        npm run server

        ** Porta setada como default 4000, caso queira alterar, alterar no arquivo package.json para a porta que você quiser.
